# 锌锭库管理系统 - 纯HTML版本

这是一个纯前端版本的锌锭库管理系统，**无需后端服务器**，数据存储在浏览器的 localStorage 中。

## 特点

- ✅ **零配置** - 双击HTML文件即可运行
- ✅ **无需安装** - 不需要Python、Node.js或任何其他环境
- ✅ **便携性强** - 可以复制到U盘随处使用
- ✅ **数据持久化** - 数据保存在浏览器本地存储中

## 文件说明

| 文件 | 说明 |
|------|------|
| `login.html` | 登录/注册页面 |
| `dashboard.html` | 主管理界面（出库管理、高铝锌锭管理） |
| `gi_inbound.html` | GI锌锭入库管理页面 |

## 使用方法

1. 双击 `login.html` 打开登录页面
2. 首次使用需要先注册账号
3. 登录后进入管理界面

## 注意事项

⚠️ **数据存储限制**：
- 数据存储在浏览器的 localStorage 中
- 清除浏览器数据会导致数据丢失
- 不同浏览器之间数据不共享
- 不同电脑之间数据不共享

## 建议

- 定期使用"导出数据"功能备份重要数据
- 如需多人协作或数据共享，请使用后端版本

## 功能列表

- 用户注册、登录、密码重置
- GI锌锭入库/出库管理
- 高铝锌锭管理
- 按年月/班组/锌锭种类统计
- 日期筛选
- 分页显示
- 数据导出（CSV格式）
- 库存校验（出库时检查库存是否充足）
